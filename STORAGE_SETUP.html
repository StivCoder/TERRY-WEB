🖼️ Image Upload Setup Guide

Complete these steps to enable image uploads to Supabase Storage

Quick Checklist

Before troubleshooting, verify you've completed these:




Step 1: Create Storage Bucket

1
Go to your Supabase Dashboard
supabase.com/dashboard →
2
Navigate to Storage
In the left sidebar, click Storage
3
Create New Bucket
Click the "New Bucket" or "Create bucket" button
4
Configure Bucket
Name: product-images
Privacy: Make it PUBLIC ✅
Images won't load if this is private
5
Save & Create
Click the Create Bucket button

Step 2: Set Up Storage Policies (RLS)

Important! Even if the bucket is public, you need RLS policies to allow uploads.

1
Go to SQL Editor
In Supabase Dashboard, click SQL Editor in the left sidebar
2
Create New Query
Click "New Query" button
3
Copy & Paste This SQL
-- Enable RLS on storage.objects ALTER TABLE storage.objects ENABLE ROW LEVEL SECURITY; -- Drop old policies if they exist DROP POLICY IF EXISTS "Public read images" ON storage.objects; DROP POLICY IF EXISTS "Public upload images" ON storage.objects; DROP POLICY IF EXISTS "Admin delete images" ON storage.objects; -- Allow ANYONE to view/download images CREATE POLICY "Public read images" ON storage.objects FOR SELECT TO anon, authenticated USING (bucket_id = 'product-images'); -- Allow ANYONE to upload images (no login required) CREATE POLICY "Public upload images" ON storage.objects FOR INSERT TO anon, authenticated WITH CHECK (bucket_id = 'product-images'); -- Allow authenticated users to delete their images CREATE POLICY "Admin delete images" ON storage.objects FOR DELETE TO authenticated USING (bucket_id = 'product-images' AND auth.uid() IS NOT NULL);
4
Run the Query
Click the blue "Run" button (or press Ctrl+Enter)
5
Verify Success
You should see a success message at the bottom. ✅

Step 3: Test Image Upload

✅ Setup Complete!
You can now test uploading images in the Admin Dashboard.
1
Go to Admin Dashboard
Open Admin Dashboard →
2
Add a New Product
Click "Add Product" button
3
Upload Product Image
Click the upload area and select an image (JPG, PNG, WebP)
Max file size: 10MB
4
Fill Product Details & Save
Complete the form and click "Save & Publish Product"

❓ Troubleshooting

Upload fails with "404 - Bucket not found"

  • ✓ Confirm bucket name is exactly "product-images" (lowercase, no spaces)
  • ✓ Check the bucket exists in Storage → Buckets list
  • ✓ Make sure bucket is set to PUBLIC

Upload fails with "Permission Denied"

  • ✓ Go back and re-run the SQL policies (Step 2)
  • ✓ Check bucket is set to PUBLIC (not private)
  • ✓ Verify "Public upload images" policy exists in Storage → Policies

Image uploads but doesn't appear on store

  • ✓ Hard refresh the store page (Ctrl+Shift+R on Windows)
  • ✓ Check image URL is correctly saved in product record
  • ✓ Verify bucket is PUBLIC (images won't load if private)

Bucket created but still says "not found"

  • ✓ Check project URL in supabase-config.js matches your Supabase project
  • ✓ Check your ANON_KEY is correct (from Settings → API Keys)
  • ✓ Try refreshing the admin page (Ctrl+R) and uploading again

📋 Common Issues & Solutions

Issue Solution
Admin shows "Local Storage" mode Your Supabase credentials aren't loaded. Check supabase-config.js has correct URL & key
Upload progress bar appears then disappears File uploaded but URL retrieval failed. Bucket might be private
Successfully uploaded but image URL is wrong Check bucket is PUBLIC in Supabase Dashboard
Want to delete old images? Go to Supabase Dashboard → Storage → product-images and delete files manually
